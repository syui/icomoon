## Origin Icons

```bash
# linux
$ git clone https://github.com/syui/ricomoon
$ mkdir -p ~/.local/share/fonts
$ cp ./ricomoon/font/icomoon.ttf ~/.local/share/fonts
$ fc-cache -fv

# use
$ open ./ricomoon/demo.html
```
